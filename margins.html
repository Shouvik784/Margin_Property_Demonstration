About Margins:

Margins create space outside an element's border, unlike padding which creates space inside the border.

Key points about margins:

  • Positive margins push other elements away
  • Negative margins can pull elements closer together
  • Using margin: auto on a block element with a defined width will center it horizontally
  • Margins can collapse between adjacent elements

Margin Shorthand Property

The margin property can take 1-4 values:

  • margin: 10px; - all four sides
  • margin: 10px 20px; - top/bottom, then left/right
  • margin: 10px 20px 30px; - top, left/right, bottom
  • margin: 10px 20px 30px 40px; - top, right, bottom, left (clockwise)
